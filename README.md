# FSE-ProyectoFinal Centro Multimedia
Proyecto final para la materia de fundamentos de sistemas embebidos. Centro multimedia en una Raspberry Pi. 

**Autores:** Hernández Alejo Ximena Gizell
             Tapia Navarro Rodrigo
**Enlace al video de demostraciòn:** https://youtu.be/FZqiHrxtxT4


## Instalación:
1. Clonar repositorio:  git clone https://github.com/RodrigoRoku/FSE-ProyectoFinal.git
2. Ejecutar: sudo ./configure.sh o sudo bash configure.sh
3. El sistema se reinicia y funciona automáticamente

## Estructura:
- src/ - Código fuente principal
- config/ - Script de configuración automática
- vid/ - TXT con ulr al video demostracion
